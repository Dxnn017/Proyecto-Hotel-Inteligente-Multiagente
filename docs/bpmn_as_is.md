# Proceso Actual (AS-IS)

## Descripción del proceso manual
Actualmente, el hotel maneja sus procesos de forma manual o semiautomatizada, lo que genera cuellos de botella y errores humanos.

### Procesos
1. **Reserva:** El huésped llama o envía un correo. Un recepcionista verifica en un Excel o PMS rudimentario si hay habitaciones. Se pueden generar sobreventas.
2. **Check-in:** A la llegada, el huésped hace fila. Se validan sus documentos manualmente, se llena una ficha en papel y se entrega llave física.
3. **Atención:** El huésped llama a recepción para pedir limpieza o quejarse. El recepcionista anota en un post-it o cuaderno y llama al área correspondiente.
4. **Facturación:** Al salir, se recolectan los vales de restaurante, frigobar, etc. Se suma manualmente o se ingresan uno a uno al sistema. Demora mucho.
5. **Feedback:** Se deja una tarjeta en la habitación que rara vez se lee. Las quejas llegan tarde.

## Problemas detectados
- Tiempos de espera largos (filas).
- Falta de personalización.
- Errores de facturación y sobreventas.
- Poca capacidad para escalar solicitudes complejas.
